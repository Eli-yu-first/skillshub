# 软著申报材料汇总清单：AI技能中心平台 V1.0

本文档旨在提供“AI技能中心平台 V1.0”（SkillsHub）在申请计算机软件著作权时所需的材料清单及申请表填写指引，以确保申报过程的顺利进行。

## 一、 核心提交纸质材料清单

以下是您需要打印并提交给中国版权保护中心的全部纸质文件：

1.  **《计算机软件著作权登记申请表》**
    -   说明：在中国版权保护中心官网在线填写后，系统会自动生成PDF文件，请将其打印并由著作权人签章。

2.  **软件源代码鉴别材料**
    -   对应文件：`软件源代码鉴别材料.docx`
    -   内容：包含本软件前30页和后30页的源代码，共60页，每页不少于50行。已按规范排版，可直接打印。

3.  **软件说明文件（文档鉴别材料）**
    -   对应文件：`软件说明文件.docx` (本文档生成器会将功能说明书、设计说明书、操作说明书合并于此)
    -   内容：包含软件的功能说明、设计思路、操作流程等，用于证明软件的完整性和功能性。已按规范排版，可直接打印。

4.  **著作权人身份证明文件**
    -   **个人申请**：提交个人身份证（正反面）复印件，并亲笔签名。
    -   **企业申请**：提交企业营业执照副本复印件，并加盖公章。

## 二、 申请表关键信息填写指引

在官网填写申请表时，请参考以下信息：

-   **软件全称**：AI技能中心平台
-   **软件简称**：SkillsHub
-   **版本号**：V1.0
-   **开发完成日期**：【请在此填写您期望的开发完成日期，例如：2026年03月01日】
-   **首次发表日期**：【若未发表，则此项不填。若已上线，则填写首次上线日期】

-   **软件作品说明**：
    ```
    本软件是一个集AI技能发现、创建、分享和管理于一体的综合性社区与工具平台。核心功能包括：1. 技能市场：用户可在此浏览、搜索和筛选由社区贡献的各类AI技能。2. 技能创建与编辑：提供标准化的在线工具，支持用户定义技能参数、编写代码逻辑并发布自己的AI技能。3. 技能详情与演练场：每个技能均有详细的文档说明，并内嵌一个交互式测试环境，用户无需集成即可实时调用并验证技能功能。4. 用户与团队空间：支持个人管理名下技能和收藏，并提供团队协作功能，允许团队成员共享和共同开发私有技能。本平台致力于降低AI技术的使用门槛，构建一个连接AI能力提供方与需求方的协作生态。
    ```

-   **技术特点**：
    ```
    开发语言：TypeScript
    开发框架：Next.js, React
    运行平台：服务器端运行于Node.js环境，客户端通过标准网页浏览器访问。
    技术架构：采用基于Next.js的全栈开发框架。利用其App Router和Server Actions特性，在单一代码库中实现了高效的前后端协作。前端UI由React Server Components和Client Components组合渲染，后端业务逻辑通过Server Actions处理。数据库采用MySQL，通过Drizzle ORM进行类型安全的数据操作。
    ```
