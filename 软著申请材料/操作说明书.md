# 操作说明书：AI技能中心平台 V1.0

## 1. 软件运行环境

本系统为B/S架构的Web应用，用户端无需安装任何特定软件，只需具备以下环境即可访问：
- **硬件环境**：PC机或笔记本电脑。
- **软件环境**：Google Chrome, Microsoft Edge, Safari等现代网页浏览器。
- **网络环境**：可访问互联网。

## 2. 系统登录与访问

- **步骤1**：在浏览器地址栏输入系统访问URL。
- **步骤2**：进入系统主页，点击页面右上角的“登录”按钮。
- **步骤3**：在弹出的登录页面中，输入您的用户名和密码，或选择通过第三方账号（如GitHub）进行授权登录。
- **步骤4**：登录成功后，系统将自动跳转回主页，右上角会显示用户的头像和昵称。

## 3. 主要功能操作流程

### 3.1. 发现并测试一个AI技能

- **步骤1：浏览技能市场**
  - 访问平台主页，即进入技能市场。用户可以通过滚动页面浏览所有技能，也可以使用左侧的分类栏（如“文本生成”、“图像处理”）或顶部的搜索框来快速定位感兴趣的技能。

- **步骤2：查看技能详情**
  - 点击任意一个技能卡片，即可进入该技能的详情页面。
  - 页面默认展示该技能的“概览”标签页，其中包含了由技能创建者编写的详细使用说明（README.md）。

- **步骤3：在线测试技能**
  - 点击切换到“演练场”（Playground）标签页。
  - 页面会根据该技能所需的输入参数，自动生成一个交互表单。
  - 用户在表单中填入测试数据（例如，对于一个“翻译”技能，在输入框中填入“Hello World”）。
  - 点击“运行”按钮。页面会在下方实时显示出该技能的执行结果（例如，显示出“你好，世界”）。

> **【主功能界面截图说明】**

### 3.2. 创建并发布一个新技能

- **步骤1：进入创建页面**
  - 登录后，点击页面右上角的用户头像，在下拉菜单中选择“创建新技能”。

- **步骤2：填写技能信息**
  - 在技能创建页面，首先需要填写技能的基本信息，如**技能名称**、**技能ID**（唯一的标识符）、**一句话简介**和**分类标签**。

- **步骤3：编写技能代码**
  - 在页面的在线代码编辑器中，使用TypeScript编写实现技能核心逻辑的函数。该函数需要接收定义的输入参数，并返回处理结果。
  - 例如，创建一个调用外部天气API的技能，代码逻辑就是发起HTTP请求并解析返回的JSON数据。

- **步骤4：编写使用文档**
  - 在Markdown编辑器中，为您的技能编写一份清晰的使用说明书，告诉其他用户这个技能是做什么的、如何使用、输入输出是什么。

- **步骤5：发布技能**
  - 填写完所有信息并编写好代码后，点击页面底部的“发布”按钮。
  - 系统会对您的代码和信息进行校验，成功后，您的新技能就会出现在技能市场上。

## 4. 异常情况处理

- **表单验证失败**：在创建技能时，如果技能ID格式不正确或必填项未填写，点击“发布”按钮时，对应输入框下方会出现红色错误提示，并阻止提交。
- **技能执行错误**：在“演练场”运行技能时，如果技能代码内部出现错误或调用的外部API失败，结果展示区域会显示一个红色的错误提示框，并打印出详细的错误日志，帮助创建者进行调试。
- **权限不足**：未登录用户尝试访问“创建技能”页面时，系统会自动跳转到登录页面，并在登录成功后重定向回创建页面。
